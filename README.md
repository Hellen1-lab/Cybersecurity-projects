# Cybersecurity-projects
My cybersecurity practices and projects

Cybersecurity-practical-
Summary of checkpoint (Labs) completed during cybersecurity training.

LAB 1- Introduction to Cybersecurity. Objective: To understand basic Linux commands and what they are used for. Tools Used: Kali Linux What I learned:

How to use common Linux commands in a cybersecurity environment.
Why Linux is important for security testing and analysis.
How commandline skills are useful for cybersecurity practice.


LAB 2- NETWORKING. Objective: To identify network reconnaissance tools and understand what they are used for. Tools Used: Kali Linux What I learned:

The role of reconnaissance in networking and cybersecurity.
Awareness of different tools used for gathering network information.
Why understanding a network is important before securing it.


LAB 3- ACTIVE DEFENSE Objective: To configure basic security and user control settings on a Windows Server. Tools Used: Windows Server What I learned:

How system administrators control what employees or users are allowed to do.
How limitations can be placed on user actions and access.
Why managing user permissions helps improve system security.


LAB 4- ATTACKS,THREATS, VULNERABILITIES & ARCHITECTURE I Objective: To check files for possible security threats. Tools Used: Windows Server What I learned:

The difference between attacks, threats, and vulnerabilities.
How scanning files can help detect suspicious activity.
Why regular system checks are important for security.


LAB 5- ARCHITECTURE & DESIGN II Objective: To analyze cybersecurity scenarios and answer questions based on given situations. Tools Used: Scenario-based questions What I learned:

How to think through security problems using real life situations.
How security decisions affect system design.
The importance of planning before implementing security solutions.


LAB 6- CRYPTOGRAPHY & SECURE SOLUTION IMPLEMENTATION I Objective: To encrypt and decrypt text using basic cryptographic techniques. Tools Used: Caesar Cipher, Vigenere Cipher What I learned:

How simple encryption and decryption works.
Why cryptography is important for protecting information.
How secure communication relies on encryption.


LAB 7- DATA ENCRYPTION Objective: To identify tools that can successfully encrypt and decrypt text. Tools Used: Atbash Cipher, Caesar Cipher, Vigenere Cipher What I learned:

Differences between various encryption methods.
How encryption helps protect data from unauthorized access.
Why encryption is a key part of cybersecurity.


LAB 8- Zed Attack Proxy (ZAP) Project Objective: To scan a vulnerable website and report identified security issues. Tools Used: Virtual Machine Workstation, Metasploit, DVWA, ZAP What I learned:

How web applications can be tested in a controlled environment.
How vulnerabilities can be identified using security tools.
The importance of ethical testing and proper reporting.
